# FAQ Chatbot

This is a simple FAQ chatbot using TF-IDF + cosine similarity.

## Files
- `faq.json`: Contains FAQ questions and answers.
- `chatbot.py`: Python script to run the chatbot.

## How to Run
1. Install dependencies:
   ```bash
   pip install scikit-learn nltk
   ```
2. Run the chatbot:
   ```bash
   python chatbot.py
   ```
3. Type your questions or `exit` to quit.
